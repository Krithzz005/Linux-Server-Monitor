# 🖥️ Linux Server Health Monitor

A menu-driven Bash application that monitors the health and performance of a Linux server. This project provides essential system monitoring features such as CPU, memory, disk, network, and process information, making it useful for Linux system administration and Site Reliability Engineering (SRE) tasks.

---

## ✨ Features

- 📌 System Information
- 📊 CPU Usage Monitoring
- 🧠 Memory Usage Monitoring
- 💾 Disk Usage Monitoring
- 🌐 Network Information
- 👥 Logged-in Users
- ⚙️ Running Processes
- 📄 Generate System Health Report
- 🎨 Menu-driven Interactive Interface

---

## 🛠️ Technologies Used

- Bash Shell Scripting
- Linux
- GNU Core Utilities

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Krithzz005/Linux-Server-Monitor.git
```

Navigate to the project:

```bash
cd Linux-Server-Monitor
```

Make the script executable:

```bash
chmod +x monitor.sh
```

Run the application:

```bash
./monitor.sh
```
